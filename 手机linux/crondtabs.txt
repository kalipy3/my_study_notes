1.安装crontab:
sudo apt install crontabs

2.启动服务：
sudo service cron start

3.重新载入配置：
sudo service cron reload

4.crontab -l命令，列出已有任务：
android@localhost:~/node_study$ crontab -l
32 * * * * /home/android/q.sh
